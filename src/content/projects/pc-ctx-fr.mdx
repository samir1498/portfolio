  - Cloudflare Pages

  - js-yaml
  - Git
features:
  - "Interface web avec tableau de bord animé, graphiques et recherche plein texte"
  - "16 outils MCP : plan_list, plan_show, plan_status, plan_validate, plan_set_status, plan_task_status, plan_add, plan_add_task, plan_references, roadmap_list, roadmap_show, research_list, research_show, graph, sync, setup"
  - "Plans avec tâches, critères d'acceptation et références croisées"
  - "Roadmaps qui référencent les plans pour donner la vision d'ensemble"
  - "Fichiers de recherche indexés depuis des dépôts liés, consultables depuis le CLI et le web"
  - "Graphe de dépendances entre plans, roadmaps et recherche"
  - "Validation : le CLI rejette les plans malformés avant lecture"
  - "`ctx setup` crée toute la structure en une commande"
  - "Serveur MCP pour les agents (appels d'outils directement, sans parsing)"
  - "Versionné par Git — les agents commitent leur propre progression"
  - "Synchronisation GitHub — `ctx sync` pousse et tire le contexte entre machines"
  - "3 packages npm publiés : @pc-ctx/core, @pc-ctx/cli, @pc-ctx/mcp"
  - "Zéro couplage à opencode — générique, fonctionne avec tout agent"
  - "Authentification Cloudflare Access — privé par défaut, zéro configuration"
plannedFeatures:
  - "Domaines idées et processus (boîte de réception pré-plan + manuel vivant)"
  - "Commandes CLI pour la gestion des journaux hebdomadaires"
  - "Format de fichier de progression standardisé"
