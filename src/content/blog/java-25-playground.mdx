## Flexible Constructor Bodies (JEP 513)

Before Java 25, the first statement in a constructor **had** to be `super()` or
`this()`. You couldn't validate parameters before delegating to the parent.

JEP 513 lifts that restriction:

```java
ChildClass(String name) {
    if (name == null || name.isBlank()) {
        throw new IllegalArgumentException("Name cannot be empty or null");
    }
    super(name);
}
```

## Scoped Values (JEP 506)

`ScopedValue` is an immutable, scope-bound alternative to `ThreadLocal`. You bind a value for the duration of a block.

<JavaWasmRunner
  id="scoped-values"
  label="JEP 506: Scoped Values"
  wasmJsUrl="/wasm-demos/scoped-values.js"
  sourceCode={`public class ScopedValues {
    private static final ScopedValue<String> CONTEXT =
        ScopedValue.newInstance();

    public static void main(String[] args) {
        ScopedValue.where(CONTEXT, "Lab_User_Session")
            .run(() -> {
                IO.println("CONTEXT = " + CONTEXT.get());
                performAction();
            });

        IO.println("Bound? " + CONTEXT.isBound());
    }

    static void performAction() {
        IO.println("Action: " + CONTEXT.get());
        nestedCall();
    }

    static void nestedCall() {
        IO.println("Nested: " + CONTEXT.get());
    }

}`}
/>

Virtual threads can have millions of concurrent scoped values
without the memory overhead of `ThreadLocal`.

## Primitive Types in Patterns (JEP 507, Third Preview)

Pattern matching (`instanceof`, `switch`) now works with primitive types.

<JavaWasmRunner
  id="primitive-patterns"
  label="JEP 507: Primitive Patterns"
  wasmJsUrl="/wasm-demos/primitive-patterns.js"
  sourceCode={`public class PrimitivePatterns {
    public static void main(String[] args) {
        testInstanceOf(100);
        testInstanceOf(100.50);

        testSwitch(42);
        testSwitch(123456L);
        testSwitch(3.14159);
    }

    static void testInstanceOf(Object obj) {
        if (obj instanceof int i)
            IO.println("int: " + i);
        else if (obj instanceof double d)
            IO.println("double: " + d);
    }

    static void testSwitch(Object obj) {
        switch (obj) {
            case int i when i > 100 ->
                IO.println("Large int " + i);
            case int i ->
                IO.println("Small int " + i);
            case long l ->
                IO.println("Long " + l);
            case double d ->
                IO.println("Double " + d);
            default ->
                IO.println("Unknown");
        }
    }

}`}
/>

## Java 25 On-Ramp (JEP 511 + 512)

Module import declarations, implicitly declared classes, and instance main methods. Combined,
you can write small programs without the usual boilerplate.
