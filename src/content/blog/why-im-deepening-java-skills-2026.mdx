Java is my favorite language.  
I’m going deeper this year, on purpose.

I don’t want feature-list knowledge.  
I want production-grade intuition.

## What I’m Studying

- Java 8 -> 25 changes with real engineering impact.
- JVM internals: class loading, JIT, GC behavior.
- Concurrency patterns, including virtual threads.
- Profiling and benchmarking before making claims.

## Execution Rule

I start with **1 hour/day**.

- Read one focused topic.
- Run one small experiment.
- Write one short summary.

## Why Public

This blog is my public log.

- What worked.
- What failed.
- What changed in my approach.

Short posts. Clear takeaways. Consistent progress.
